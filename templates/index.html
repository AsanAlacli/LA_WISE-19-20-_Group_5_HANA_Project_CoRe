Course Recommendation Guide


This platform helps you to see your grade based on capability and interest.


How to start:

  • In Home you can find information about how to work with this platform
  • In Survey you can give requested information about yourself and see the predicted grade of desired course
  • In Analytic there are three kind of visualization of our data set, the first one is Bar chart based on "Mean of grade based on attending courses between males and females". The second chart is Scatter plot which is user interactive. User select his/her study program and see the result based on age and grade. The third chart is Pie plot which user can see percentage of some important features in our dataset, like gender, age, course name , language, previous knowledge, study time per week
  • In About you can find our contribution
